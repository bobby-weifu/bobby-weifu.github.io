            • 3.数学工具
              • mMass 提供许多常见数据处理功能：
              • 数学操作，裁剪，基线校正，精加工，峰值挑选， 去同位素，反褶积，批处理，数据交换等（为加快来自不同仪器的数据处理，可定义用户处理预设）
              • 3.1 数学操作（对应title工具栏Processing→Math Operations）
                • 可用于标准化，合并，剔除，叠加文档（大多数操作只能在移除峰值列表时应用于质谱点）
              • 3.2裁剪（Processing → Crop ）
              • 3.3基线校准（Processing → Smooth Spectrum）
                • 特别对于蛋白质的MALDI_TOF质谱来说，较强的基线畸变是一个普遍存在的问题，需要经常进行校正。
                • 基线是根据质谱噪声作为所有数据点的中位数来计算。
                • 操作：使用精密滑块来设置基线，以精确跟踪质谱形状。
                • 精度-基线形状精度（1=直线）
                • 相对偏移-基线偏移
              • 3.4平滑处理（Processing → Smooth Spectrum）
                • 可以使用这个函数来处理扭曲峰值形状的噪声。
                • 常规三种滤波平滑算法：
                • Moving Average（滑动平均线）
                • Gaussian（高斯线）
                • Savitzky-Golay（S-G滤波器）
              • 3.5去同位素（Processing → Deisotoping）
                • 主要用途为去除峰值挑选后不需要的峰同位素。由于，为了找到同位素，需要计算一个峰值电荷，因此该工具也可以用来测定电荷。
              • 3.6峰值挑选（Processing → Peak Picking）
                • mMass加入了自动峰值挑选算法，以区别于部分繁琐的手动峰值标记工具（FTICR MS, Orbitrap）
              • 3.7 反褶积（Processing → Deconvolution ）
              • 3.8 批处理（Processing → Batch Processing）
                • 可以使用批处理工具一次性对多个文档应用进行相同的处理步骤。
              • 3.9 数据交换（Processing → Swap）
                • mMass无法确定 ASCLL XY 和一些mzXML文档中是否包含质谱点或者仅包含峰的质心，因此默认只选择质谱，可自行切换。
            • 4. 数据校准（Processing → Calibration）
              • mMass支持两种校准方法：标准校准和统计校准。
              • 左上两个工具：
              • 展示校准工具栏/展示校准错误图
            • 5.元素周期表（Tools → Periodic）
            • 6.质量计算器（Tools → Mass Calculator）
              • 6.1语法公式
                • 所有已知的元素和它们的同位素都在mMass的库中定义，可以用来定义任何化合物。使用以下语法规则指定复合公式：
                  • 一，使用常用语法输入简单公式：C16H32O2
                  • 二，使用括号来定义组：CH3(CH2)14COOH
                  • 三，使用负 ‘-’来移除原子：CH3(CH2)14COOH(NaH-1)
                  • 四，使用大括号 ‘{}’来定义特定的同位素元素：C{13}16H32O2
              • 6.2化合物摘要面板
              • 6.3离子系列面板
                • 根据极性设置计算相应的单同位素和平均质量。默认情况下，质子被用作‘充电剂’，但你可以制定自己的公式或 ‘e’来表示自由基离子。
              • 6.4同位素模型面板
                • 同位素模型面板为从复合公式生成理论同位素模型提供了一个简单的工具。根据指定的峰宽（FWHM）计算同位素并将其分组。对于每个同位素。峰形可以被建模为对称（高斯）或不对称（半高斯，半洛伦兹），最终轮廓是作为所有峰的总和计算的。
                • 理论同位素分布图可与实测数据叠加，并使用FWHM，强度，基线和 m/z位移 精确定位。用户也可以将理论配置文件保存为常规文档。
            • 7.质量公式工具面板（Tools → Mass to Formula）
              • 勾选同位素模型审查
              • 合成规则选项（Composition rules）：
                • 一，H/C 规则 - 如果勾选，只显示具有一下氢/碳比率的化合物：
                • 二，NOPS/C 规则 - 如果勾选，只显示具有以下杂原子/碳比率的化合物：
                • 三， NOPS 规则 - 如果勾选，只显示具有一下杂原子数的化合物：
                • 四， RDBE 规则 - 如果勾选，只显示具有以下中性化合物的环和双键当量的化合物：
                • 五， integer RDBE 规则 - 如果勾选，只显示具有整数值的环和双键等效中性化合物，它对应于具有均匀电子排布的中性化合物。
            • 8.质量缺陷图（Tools → Mass Defect Plot ）
            • 9.内部序列编辑器
              • 9.1 序列编辑器（+ → Sequence ）
                • 说明：此工具可以定义蛋白质或肽序列共其他工具使用。尽管典型的蛋白质序列由规则的蛋白原氨基酸组成，并进行了一些可选的修饰，但对于天然肽的数量却并非如此，其中也使用了各种结构单元（单体）。因此，mMass提供了两种不同的序列编辑器，可以通过“序列类型”(sequence type)组合框进行切换。 此外，序列结构可以设置为线性或循环。
                • 9.1.1 标准序列编辑器
                  • 标准系列编辑器中，序列可以手动输入，从剪贴板粘贴或从任何mMass 或 FASTA文档导入。每个输入都被自动审查为只包含常规氨基酸。使用修改面板进行任何修改，都会高亮修改后的氨基酸为红色。
                • 9.1.2 定制类型序列编辑器
                  • 自定义类型的序列编辑器，可以创建由多达20个用户可定义的单体组成的任何非标准肽序列，序列单体可以手动输入相应字段，或从 单体面板（Monomer Library）中删除。
              • 9.2 序列修饰面板
                • 在序列修饰面板中，任何转义修饰都可以设置为序列末端，单个氨基酸或选定的类型。在位置旁边，用户可以为每个修饰指定是固定还是可变的，然后所有可变体在序列相关的工具中被计算。为了避免永久分配相同的修饰集，您可以定义自己的预设并经常使用。
              • 9.3 蛋白质消化
              • 9.4. 肽碎片
              • 9.5 数据匹配
              • 9.6 质量搜索
                • 用户可以通过质量搜索工具查询到序列中指定 m/z 的任何肽段，该工具对于鉴定由蛋白质的非特异性或半特异性裂解产生的峰可能非常有用。

    3.5去同位素（Processing → Deisotoping）

    主要用途为去除峰值挑选后不需要的峰同位素。由于，为了找到同位素，需要计算一个峰值电荷，因此该工具也可以用来测定电荷。
    从指定的最大电荷量开始，用对应的同位素质量差（ 1.OO287/abs ( z ) ）±同位素质量容限来寻找每个峰的同位素。如果至少发现一个同位素，则将母峰设为具有当前电荷状态的单同位素。如果没有发现同位素，则降低当前电荷状态（ abs ( z ）-1 ) ，重新开始寻找同一峰。由于可能的峰重叠，需要考虑理论同位素模式。对各同位素的强度与理论值进行了比较．如果强度与理论值士同位素强度公差匹配，则在后续的搜索周期中丢弃相应的同位素。如果差值过大，则在后续的搜寻周期中使用相应的同位素作为可能的母质（单同位素）峰。由于 mMass 主要用于蛋白质组学数据解释，我们使用平均值（ C4.9384 N1.3577 O1.4773 S0.0417 H7.7583 ）来近似理论同位素模式（ Senko et al．1995 AJMS ）。

    去同位素功能参数 相关功能说明
    Maximum charge 设置需要搜索的最大电荷（电荷：离子的电流电荷和电离物质）
    Isotope mass tolerance 同位素质量公差：对相邻同位素之间质量差的公差
    Isotope intensity tolerance 同位素强度公差：理论同位素和实测同位素强度检查的公差
    Isotope mass shift 同位素质量位移：对使用的默认同位素距离（1.00287）进行质量校正
    Remove isotopes 去除同位素：所有确认的同位素将在处理后去除
    Remove unknown 去除未知峰：所有未知峰将在处理后被去除
    Label envelope tool
    Envelope intensity Envelope 强度：用于计算 Envelope强度的方法
    Set labels as monoisotopes 设置标签为单同位素：手动标记的峰自动设置为单同位素

    3.6峰值挑选（Processing → Peak Picking）

    mMass加入了自动峰值挑选算法，以区别于部分繁琐的手动峰值标记工具（FTICR MS, Orbitrap）
    自动峰值挑选算法基本步骤：

 一，基线校准-如果启用：基线和噪声宽度会被计算并用于每个峰值。

 二，平滑处理-如果启用：原始质谱会被暂时平滑处理以消除峰值噪声。

 三，搜索局部最大值：局部最大值被暂时标记为潜在峰。

 四，阈值过滤：低于或高于用户设置的裁剪阈值区间的峰值被剔除。

 五，质心计算：计算峰的质心并将未解析的峰分组在一起。

 六，移除肩峰-如果开启：FTMS数据的‘肩峰’会被移除。

 七，去同位素-如果启用：计算峰电荷并去同位素。

    自动峰值挑选功能参数 功能说明
    S/N threshold 信噪比阈值：低于信噪比阈值的峰值不会被标记。
    Abs. intensity threshold 绝对强度阈值：低于绝对强度（峰值强度-基线）阈值的峰值将不被标记。
    Rel. intensity threshold 相对强度阈值：低于相对强度阈值的峰值将不被标记
    Picking height 选取高度：质心的选取高度
    Apply baseline 每个峰值的基线和噪声宽度根据当前基线设置计算。
    Apply smoothing 根据当前的平滑设置，数据会在峰值选取前被平滑处理。
    Apply deisotoping 应用去同位素：根据当前的去同位素设置，峰同位素将根据电流计算出电荷，然后去同位素。
    Remove shoulder peaks 移除肩峰：展示在FTMS数据中的小周围峰（肩峰）将被移除。

    3.7 反褶积（Processing → Deconvolution ）

    ESI 质谱通常包含不同电荷态形式的同种物质，这使数据解释复杂化。简单过程经常被用来垂新计算所有多重电荷的物种为单一电荷的形式。此外，对于来自同一物种但电荷态不同的相关峰，可以使用己知峰的半峰宽（FWHM) 值或默认的 m/z 窗口将其组合在一起。通过应用反褶积，用反褶积峰表生成当前文档的副本。

    反褶积功能参数 功能说明
    Mass type 质量类型：提供重新计算电荷的标准质量类型
    Grouping window 窗口分组：将未知半峰宽值的峰值进行分组
    Group peaks 邻近分组：将分组应用于相邻的峰值
    Force grouping window 强制分组：即使已知FWHM值，依然使用默认分组窗口

    4. 数据校准（Processing → Calibration）

    mMass支持两种校准方法：标准校准和统计校准。
    标准校准：
  首先选择任意参考列表，并将参考值分配给测量的峰，参考值使用公差值自动分配。用户可以使用线性或二次拟合来计算校准常数。一般来说，用户想要在指定的参考范围外重新校准 m/z 范围，线性拟合会更加安全。
统计校准：
  又名’自我校正‘，是肽类质谱的一种特殊方法。它无需任何外部参考点，因为它是基于缩氨酸的单同位素质量以非常有规律的方式分布这一事实。只需要将其积分部分乘以因子 1.00048 ，就可以充分精确地计算出肽质量的小数部分。对于这种方法，每个超过 700 m/z 极限的峰都重新计算并作为参考质量。

    8.质量缺陷图（Tools → Mass Defect Plot ）

    质量缺陷图是一种简单的工具，可以根据所选文档的质量分数或各种类型的质量缺陷来可视化其峰值。在生成的图中，相似的化合物倾向于聚集成簇。标准质量缺陷的定义是化合物或峰的精确质量与其名义质量之间的差。在这里，名义质量可以计算为“圆形”，“地板”（向下舍入）或“天花板”（向上舍入）。 类似地，通过将标准质量缺陷除以峰的电流 m/z 并乘以1e6，可以计算出相对质量缺陷。 Kendrick 质量缺陷可用于对所有仅在CH2基团数（或任何用户指定的式）上不同的化合物进行分组。

    Nominal mass 名义质量：用于名义质量计算的函数（精确质量舍入）
    Kendrick formula Kendrick 公式: Kendrick 质量缺陷的分子式
    Int. threshold 隐藏峰值低于指定的相对强度限制
    Remove isotopes 移除同位素: 移除指定的同位素（移除前需要操作去同位素步骤）
    Ignore charge 忽略电荷：使用获得的 m/z值，不需要重新计算电荷值
    Highlight annotated 高亮标注：高亮显示所有标注的峰和匹配序列
    Show all documents 显示所有文档：将所有可见文档覆盖在一个图中
    Plot 生成一个质量缺陷图

    9.内部序列编辑器

    mMass 提供了一个内部序列编辑器，可以使任何蛋白质或肽序列可供其他工具使用。任何修改都可以作为固定或可变应用。蛋白质文摘工具可用于生成由特定蛋白质序列的计算机化酶解或化学消化产生的多肽列表。同样，“肽片段化”工具会生成常见的一个肽片段的列表。在这两种情况下，所有可能的变量修改组合都被计算出来，结果可以很容易地与测量数据进行比较。此外，可以在序列搜索亚肽质量并通过 Mass Search 工具来识别非特异性或半特异性裂解。

    9.1 序列编辑器（+ → Sequence ）

    说明：此工具可以定义蛋白质或肽序列共其他工具使用。尽管典型的蛋白质序列由规则的蛋白原氨基酸组成，并进行了一些可选的修饰，但对于天然肽的数量却并非如此，其中也使用了各种结构单元（单体）。因此，mMass提供了两种不同的序列编辑器，可以通过“序列类型”(sequence type)组合框进行切换。 此外，序列结构可以设置为线性或循环。
    9.1.1 标准序列编辑器
    标准系列编辑器中，序列可以手动输入，从剪贴板粘贴或从任何mMass 或 FASTA文档导入。每个输入都被自动审查为只包含常规氨基酸。使用修改面板进行任何修改，都会高亮修改后的氨基酸为红色。

    9.1.2 定制类型序列编辑器
    自定义类型的序列编辑器，可以创建由多达20个用户可定义的单体组成的任何非标准肽序列，序列单体可以手动输入相应字段，或从 单体面板（Monomer Library）中删除。

    9.2 序列修饰面板

    在序列修饰面板中，任何转义修饰都可以设置为序列末端，单个氨基酸或选定的类型。在位置旁边，用户可以为每个修饰指定是固定还是可变的，然后所有可变体在序列相关的工具中被计算。为了避免永久分配相同的修饰集，您可以定义自己的预设并经常使用。
    序列修改面板功能参数 功能参数说明
    Show specific modifications only 仅显示特定的修饰：显示特定选择的氨基酸
    Position 位置：氨基酸位值以及类型
    Modification 修改：需要修饰的名称和类型
    add 添加修饰：选择氨基酸类型，位置 修饰和修饰类型并添加
    Remove 删除修饰：悬着列表中需要删除的修饰，并删除

    10 . 质量过滤器（Tools → Mass Filter）

    在许多质谱中，常见的污染物质量会对数据解释产生负面影响。使用 Mass Filter 工具，这样的污染物可以很容易地注释掉或从峰值列表中被删除。此外，匹配的峰可以作为数据重新标准的内标。

    质量过滤器面板功能参数 功能参数说明
    Tolerance 公差：用于过滤的质量公差
    Da, ppm 公差单位：道尔顿， ppm
    Ignore charge 忽略电荷：数据匹配时忽略峰值电荷
    Annotated 注释：不使用用户注释中使用峰
    Matched 匹配：不使用匹配肽段或序列片段的峰
    Unselected 未选择：仅使用在峰值列表面板中选择的峰
    Isptopes 同位素：不使用同位素
    Unknown 未知项：不使用未分配同位素编号或电荷的峰

    11.化合物检索（Tools → Compounds Search）

    该工具可辅助用户搜索任何指定的化合物，甚至是峰值列表中的化合物列表。每个化合物在 mMass 库中都指定为分子式、名称和描述，因此可以任意搜索单同位素和平均质量。此外，也可以自动搜索常见的加合物，可以注释所有匹配的峰，然后在分析报告中显示或打印

    化合物检索面板功能参数 功能参数说明
    Compounds 化合物：需要检索的化合物列表
    Formula 化合物公式：检索的化合物公式
    Mass 质量：要检索的质量类型
    Max charge 最大电荷：需要计算的最大电荷
    M* 离子：勾选计算自由基离子
    Adducts 加合物：可检索的加合物类型

    12.峰值的差异（Tools → Peak Differences）

    质朴解析通常需要没完没了地检查质谱中所有峰的差异。但是，峰值差异工具能够简单地生成一个列表，里面包含峰值列表中所有峰值之间的差异。然后，可以使用该表在指定的公差范围内自动将每个差异与所有氨基酸，计算的二肽或指定的m / z值的相应质量进行比较。因此，峰值差异工具为多肽从头测序或 PTM 检索 提供了有力的帮助。

    化合物检索面板功能参数 功能参数说明
    peak list 峰值列表
    table of differences 差异表
    matches for selected difference 匹配选定的差异
    Difference 差异：用户自定义差异检索框
    Amino acids 氨基酸：勾选以检索所有可用的氨基酸
    Dipeptides 二肽：勾选已检索所有计算出的二肽
    Mass 质量：勾选所需要的质量类型用于氨基酸和二肽计算
    Tolerance 公差：用户自定义质量公差条件
    Hide umatched 隐藏未匹配：勾选将只显示匹配的峰值

    13.峰值比较工具（Tools → Compare Peak Lists）

    与峰值差异工具类似，峰值比较工具可以很容易地用于比较多个文档之间的峰值列表或注释。使用该工具，所有文档都将以其自己的光谱颜色进行加载和标记，并且可以在指定的质量公差范围内甚至使用强度比阈值进行比较。对于不同文档中标识的每个峰值，对应的文档颜色将在 m/z 值旁边高亮显示。通过在峰列表中选择任何m / z值，将显示所有匹配的峰，然后显示误差和强度比（a-选择的峰，b-匹配的峰）。 除单个峰列表外，还提供用于组合峰列表的单独表格。

    峰值比较功能参数 功能参数说明
    individual peak lists 用户个人峰值列表
    combined peak list 结合峰值列表
    matches for selected peak 匹配选定峰值
    Compare 用于比较的数据类型，例如 peak lists
    Tolerance 公差：用于比较的质量公差
    Da, ppm 公差单位：道尔顿， ppm
    Ignore charge 忽略电荷：数据匹配时忽略峰值电荷

    14.质谱发生器（Tools → Spectrum Generator）

    在各种情况下，例如识别峰重叠或验证不同算法，生成人工质谱图与获取的数据进行比较或测试算法功能可能很有用。使用质谱发生器，用户可以将所选文档的峰列表中的每个峰建模为对称（高斯）或不对称（半高斯，半洛伦兹）曲线，并具有相应的强度，指定的宽度（FWHM）和每个峰的点数。将所有的峰组合在一起，以形成最终的轮廓质谱，它可以覆盖质谱查看器的内容或保存到选定的文件中。如果想要添加更加逼真的外观，您可以应用一些指定宽度的随机噪声。

    质谱发生器功能参数 功能参数说明
    Peak shape 峰形：对称（高斯）或不对称（半高斯，半洛伦兹）峰形
    Default FWHM 默认FWHM：如果峰值的值未知，则使用默认FWHM值
    Force 强制FWHM：即使峰值已知，依然使用默认FWHM值
    Points/peak 点/峰：绘制每个峰值的近似点
    Noise width 噪声宽度：随机噪声的宽度设定
    Show peaks 展示峰：在生成的质谱中显示单个峰的形状
    Show overlay 展示覆盖：在主质谱查看器中显示生成的配置文件
    Flip overlay 翻转叠加：在主质谱查看器中显示翻转的叠加轮廓
    Generate 生成质谱：根据当前参数生成质谱
    Apply 应用：将生成的轮廓图应用于当前文件

    15.包络拟合（Tools → Envelope Fit）

    包络拟合是确定重原子结合到分子中的简单工具，氢/氘交换实验可能是一个典型例子。 通常，它计算了特定公式和交换损益公式的所有可能的同位素模式，并使用它们的线性组合和最小二乘拟合将建模的模式拟合到获取的数据。 结果显示了每个单独状态的百分比以及平均掺入量。

    包络拟合功能参数 功能参数详解
    Formula 公式：用于建模的化合物基本公式
    Charge 电荷：需要拟合的峰的电荷状态
    Exchange 交换：需要交换的原子（损耗 vs. 增益）
    Range 范围： 交换原子的最小和最大数目
    Fit to 拟合：勾选需要拟合的数据类型 （峰列 或 质谱）
    Default FWHM 默认FWHM：如果峰值的值未知，则使用默认FWHM值
    Force 强制FWHM：即使峰值已知，依然使用默认FWHM值
    Rel. threshold 相对阈值：涉及峰的相对强度阈值（相关质量范围的基峰）
    Auto align 自动对齐： 自动将获取的数据与建模的包络对齐
    Average X 平均X：计算交换原子的平均数
