    mMass检测软件剖析

    开源质谱检测软件 Mmass工具详解

    1.安装文档说明：https://bobby-weifu.github.io/markdown/mmass/

    2.主面板布局：

    2.1 title功能介绍：

    从左到右对应功能 对应说明
    Processing tools panel 处理工具面板
    Calibration tool panel 校准工具面板
    Sequence tools panel 序列工具面板
    Periodic Table panel 元素周期表面板
    Mass Calculator tools panel 质量计算器工具面板
    Mass to Formula tool panel 质量公式工具面板
    Mass Defect Plot tool panel 质量缺陷图工具面板
    Mass Filter tool panel 质量过滤器工具面板
    Compound Search tool panel 复合搜索工具面板
    Peak Differences tool panel 峰值差异工具面板
    Compare Peak Lists tool panel 比较峰列表工具面板
    Spectrum Generator tool panel 频谱发生器工具面板
    Envelope Fit tool panel Envelop Fit工具面板
    Mascot Search tools panel Mascot标准质谱搜素工具面板
    ProFound Search tool panel Profound标准质谱搜索工具面板
    Document Info panel 文档信息面板
    Analysis Report 导出html分析报告
    Export tools panel 导出文本信息

    软件使用实验消解中使用的相同酶对数据库蛋白质进行计算机内消解，并生成理论峰列表。 Mascot，MS-Fit和Profound是最常用于PMF的搜索程序。

    2.2文件面板：

    主要目的是提供已加载文档及其所有资源素的结构视图。

    从上到下面板信息 文件面板说明
    spectrum colour and visibility bullet 光谱颜色和可见性项目符号
    document title 文件名
    user annotations 用户注释
    sequence title 序列标题
    peak m/z, (match error) and annotation 峰值m / z，（匹配误差）和注释
    sequence matches 序列匹配
    delete item 删除项目
    add blank document or sequence 添加空白文档或序列

    2.7文档导出Web html （Report）

    2.8文档信息（Notes and information）

    文档信息面板栏 文档信息栏说明
    首栏（文档摘要） 可以指定文件标题，操作人员姓名，联系人和机构，使用的仪器，测量日期等，并可以设置预设值
    中栏（频谱信息） 可以指定扫描ID，保留时间，MS级前驱体m/z，前驱电荷和极性
    尾栏（分析笔记） 可以输入关于样品制备，样品分析，和解释结果的笔记
